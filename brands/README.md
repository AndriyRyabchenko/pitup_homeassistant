# Іконки для home-assistant/brands

Щоб PitUp мав іконку в HACS та на сторінці інтеграцій HA, ці файли треба додати
у репозиторій [home-assistant/brands](https://github.com/home-assistant/brands)
за шляхом `custom_integrations/pitup/` (окремим PR):

- `icon.png` — 256×256, прозорий фон
- `icon@2x.png` — 512×512
- `logo.png` — 512×512

Після мержу PR іконка з'явиться автоматично (може кешуватись до кількох годин).
